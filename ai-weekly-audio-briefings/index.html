Project overview

Automated weekly scan of seven top AI publications. An agentic workflow curates, summarizes, and narrates updates with GPT, organized by five business criteria. Outputs a podcast with transcript and citations.

Business impact

Keeps executives current by turning complex AI news into short audio briefs, cutting screen time and boosting retention, ideal for commutes.

My role: Strategy → architecture → build → test

Designed and shipped the agentic workflow: GPT summarization, ElevenLabs TTS, and Make automations. Built data collection, summarization, and publishing logic and validated the full pipeline to RSS.

Scalability

Flexible: vary topics, cadence, and length for finance, healthcare, education, or policy; support regional versions, sponsor reads, and multi-language feeds.

Technical details

OpenAI ElevenLabs Make / n8n GitHub Pages RSS / feed.xml GitHub Actions
  • OpenAI (GPT) – condenses articles and drafts a natural, listenable script
  • ElevenLabs TTS – generates narration as MP3
  • Make / n8n – automates scraping, summarization, TTS, and publishing
  • GitHub Pages + RSS (feed.xml) – hosts MP3s and enables podcast subscriptions
  • GitHub Actions (optional) – automates weekly builds and publishing
