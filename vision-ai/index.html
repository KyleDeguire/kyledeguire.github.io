VisionAI: Emotion Detection

Overview

What it is:

A mobile Vision AI app that analyzes facial expressions in real time to classify emotions using an optimized TensorFlow Lite model in SwiftUI. Built for fast, private, on-device inference without an internet connection.

Why it matters:

Gives UX and product teams a quick, privacy-safe way to capture emotion insights for personalization and interface improvement.

My role:

Strategy → architecture → build → test
Trained the emotion-classification model using FERPlus and RAF-DB datasets, optimized it for TensorFlow Lite, and integrated live camera inference in SwiftUI.

Scalability:

Extends to real-time sentiment tracking, adaptive UX feedback, and integrations with AR/VR or education use cases.
