VisionAI: Emotion Detection demo

Deep learning model that predicts facial emotion from images. Trained with TensorFlow and Keras, exported to TensorFlow Lite for iPhone deployment in Xcode. OpenCV used for preprocessing and detection during development.

Tools: TensorFlow, Keras, OpenCV, NumPy, Pandas, TensorFlow Lite (TFLite), Xcode

Examples

Each tile shows the model’s top prediction and confidence.

Notes

  • Images can be replaced at any time by uploading new files to this folder with the same names.
  • If you use different filenames, update the src="..." in each <img> tag to match.
