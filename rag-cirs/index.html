Project overview

A retrieval-augmented assistant for FAA Advisory Circulars. PDFs are parsed, cleaned, chunked, and embedded into Pinecone; a LangChain retriever routes queries to OpenAI and returns cited answers with source passages.

Business impact

Helps aviation teams find and interpret regulations faster with cited, readable summaries that improve audit readiness and decision confidence.

My role: Strategy → architecture → build → test

Built the ingestion pipeline for AC PDFs, text cleaning and chunking, embeddings to Pinecone, and a LangChain QA chain with citations. Added evaluation spot-checks and a lightweight demo front end.

Scalability

Extend across airline operations, maintenance manuals, MELs, training programs, and regulatory reporting—or adapt to other compliance-heavy domains (finance, healthcare, energy).

Technical details

OpenAI LangChain Pinecone PDF ingestion Chunking + Embeddings Citations
  • Ingestion – parse FAA Advisory Circular PDFs, clean text, and normalize metadata
  • Chunking – split with structure-aware rules to preserve headings and context
  • Embeddings → Pinecone – store vectors with doc and section metadata
  • Retriever + QA chain – LangChain pipeline queries vectors, composes a cited answer
  • Evaluation – spot checks with held-out questions to tune chunk size and prompts
  • Front end – lightweight demo surface with answer + sources view
