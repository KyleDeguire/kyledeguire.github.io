Project overview

Retrieval-augmented assistant for FAA Advisory Circulars. Documents are parsed, cleaned, chunked, and embedded in Pinecone; a LangChain retriever routes queries to OpenAI and returns cited answers with source snippets.

Business impact

Helps aviation teams find and interpret regulations faster with cited, readable summaries that improve audit readiness and decision confidence.

My role: Strategy → architecture → build → test

Built the ingestion pipeline for AC PDFs, text cleaning and chunking, embedding to Pinecone, and a LangChain QA chain with citations. Added evaluation spot-checks and a light demo interface.

Scalability

Extend across airline ops, manuals, training, and regulatory reporting—or adapt to any compliance-heavy domain (finance, healthcare, energy).

Technical details

  • Ingestion – parse FAA Advisory Circular PDFs and normalize metadata.
  • Chunking – split text using structure-aware rules to preserve context.
  • Embeddings → Pinecone – store vectors with document and section metadata.
  • Retriever + QA chain – LangChain queries vectors and composes cited answers.
  • Evaluation – spot check accuracy to refine chunk size and prompt design.
  • Front end – lightweight demo surface with answer + source display.
