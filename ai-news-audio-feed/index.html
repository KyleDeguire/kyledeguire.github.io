AI Weekly Audio Feed

Overview

Automated Weekly RSS-ready GPT + ElevenLabs

Project overview

Automated news-to-audio pipeline that curates, summarizes, scripts, and narrates weekly AI updates. Produces MP3 episodes and an RSS feed for publishing on GitHub Pages and podcast apps.

My role: Strategy → architecture → build → test

Implemented source collection, GPT summarization and script generation, ElevenLabs TTS, and automated publishing to GitHub Pages RSS.

Business impact

Keeps busy professionals informed in minutes by turning complex AI news into concise, high-quality audio briefs.

Scalability

Highly flexible — change subject matter, sub-topics, cadence, and episode length for domains like finance, healthcare, education, or policy.

Technical details

OpenAI ElevenLabs Make / n8n GitHub Pages RSS / feed.xml GitHub Actions
  • OpenAI (GPT) — condenses sources and drafts a natural, listenable script
  • ElevenLabs TTS — generates clear narration as MP3
  • Make (Integromat) / n8n — orchestrates scraping, summarization, TTS, and file moves
  • GitHub Pages — hosts MP3 files and a public feed.xml
  • RSS — enables podcast client subscriptions (Apple/Overcast/etc.)
  • GitHub Actions (optional) — schedules weekly builds and publishes artifacts
